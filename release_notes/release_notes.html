PagerDuty Release Notes - Published by Splunk January 25, 2022

Version 2.0.6 - Released January 25, 2022
  • Marked the app as FIPS Compliant [PAPP-22949]
Version 2.0.3 - Released December 18, 2020
  • Compatibility changes for Python 3 support
Version 1.0.27 - Released May 16, 2019
  • Added support for the Pagerduty API v2
  • Removed get_oncall action
  • Implemented pagination in all the 'List' actions
  • Bug fixes in Create incident and List policies actions
Version 1.0.21 - Released September 21, 2018
  • Added new 'get oncall user' action that returns the user for a specific escalation policy
  • Added new 'get user info' action that returns the user info given a user id
Version 1.0.19 - Released May 14, 2018
  • Added new action, "create incidents"
  • Added new action, "list oncall" and deprecated "get oncall" action
Version 1.0.14 - Released February 06, 2018
  • App action views and Logo updates
Version 1.0.9 - Released August 23, 2017
  • 3.0 compatibility update release only
Version 1.0.7 - Released July 27, 2017
  • 3.0 compatibility update release only
Version 1.0.6 - Released October 14, 2016
  • Made improvements to the datapath settings
Version 1.0.5 - Released August 15, 2016
  • Added a contains type to the email addresses
